### Built and tested on Ubuntu 24.04.2 LTS

To compile and run the project, follow these steps in your terminal:


1. Create a build directory
```bash
mkdir build
cd build
```


2. Generate Makefiles with CMake
```bash
cmake ..
```
3. Compile the project
```bash
make
```
4. Run the program
```bash
./to_Do
```
